# Project Design Management Rules

- **Design Guideline Compliance**: Always refer to and strictly follow [Design.md](file:///Users/kji96/Projects/SearchCompany/Design.md) in the root directory for all UI/UX styling, color schemes, typography, layout structures, and component designs.
- **Reference Designs**: Utilize the sample code and layout structures in the [_sample_design](file:///Users/kji96/Projects/SearchCompany/_sample_design) folder as reference templates to align the visual hierarchy and design details of the frontend dashboard and modals.
- **Local Dev Server Execution**: When the user requests to check things locally, always execute `npm run dev` in the `frontend` folder, wait for the server to start, and provide the local access link (e.g., `http://localhost:3000` or the actual assigned port if 3000 is occupied).
