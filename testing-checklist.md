# 테스트 체크리스트

## 기능 테스트 (기업 등록 & 지원사업 등록 공통)

### 1. 파일 다운로드
- [ ] "Excel 다운로드" 버튼 클릭 시 정상 다운로드
- [ ] 다운로드된 파일 확장자가 `.xlsx` 임
- [ ] 다운로드된 템플릿 파일의 컬럼 헤더 확인
  - 기업 등록: 기업명, 사업자등록번호, 소재지, 주요 사업(지원분야), 주요 제품
  - 지원사업: 사업자등록번호, 연도, 지원사업명, 지원과제명, 상태, 선정금액, 지원금액, 비고

### 2. 파일 업로드
- [ ] 파일 선택 영역 클릭 시 파일 탐색기 열림
- [ ] 드래그 앤 드롭 지원
- [ ] 지원 파일 형식: .xlsx, .xls, .csv
- [ ] 잘못된 파일 형식 업로드 시 적절한 오류 메시지

### 3. 파싱 결과 미리보기
- [ ] 파일 선택 후 자동으로 파싱 시작
- [ ] 로딩 인디케이터 표시 중
- [ ] 파싱 결과가 테이블 형태로 표시
- [ ] 테이블 헤더가 해당 모드에 맞게 표시
- [ ] 행 번호 컬럼 존재
- [ ] 데이터 정렬 및 정렬 옵션 확인 (선택사항)

### 4. 데이터 검증 및 상태 표시
- [ ] 필수 필드 누락 행에 경고 표시
- [ ] 기업 등록: 기업명 또는 사업자번호 누락 시
- [ ] 지원사업: 연도, 지원사업명, 상태 중 하나라도 누락 시
- [ ] 매칭 실패 행 표시 (지원사업 전용): "(매칭 기업 없음)"
- [ ] 중복 행 표시 (기업 등록 전용): "신규", "중복(업데이트)", "중복(건너뛰기)"
- [ ] 수치형 필드에 문자 입력 시 적절한 변환 또는 경고

### 5. 업로드 수행
- [ ] "업로드 진행" 버튼 클릭 시 활성화 (유효한 데이터가 있을 때)
- [ ] 진행 바 또는 로딩 스피너 표시
- [ ] 취소 기능: 업로드 중 "취소" 버튼으로 중단 가능 (가능한 경우)
- [ ] 동일 파일 재업로드 방지 메커니즘 확인

### 6. 결과 피드백
- [ ] 성공 시 토스트 알림: "◯◯ 등록이 완료되었습니다." (기업/지원사업 구분)
- [ ] 실패 시 토스트 알림: 구체적인 오류 메시지 표시
- [ ] 부분 성공 시: 성공 건수, 실패 건수, 실패 원인 요약 표시
- [ ] 업로드 완료 후 자동으로 모달 닫힘 또는 유지 옵션

## 기업 등록 전용 테스트

### 1. 중복 처리
- [ ] 기존 사업자번호와 동일한 데이터 업로드 시
- [ ] 미리보기에서 "중복(업데이트)" 또는 "중복(건너뛰기)" 표시 옵션 제공
- [ ] 선택에 따라 적절히 동작 (업데이트/건너뛰기/신규 취급)

### 2. 필수 필드 검증
- [ ] 기업명 누락 시 행 제외 및 경고
- [ ] 사업자번호 누락 시 행 제외 및 경고
- [ ] 사업자번호 형식 검증 (숫자+하이픈)

## 지원사업 등록 전용 테스트

### 1. 기업 매칭 테스트
- [ ] 존재하는 사업자번호 → 매칭 성공, company_id 획득
- [ ] 존재하지 않는 사업자번호 → "(매칭 기업 없음)" 표시, 행 제외
- [ ] 사업자번호 형식 다양성 테스트 (하이픈 유무, 공백 포함 등)

### 2. 데이터 매핑 정확도
- [ ] 사업자등록번호(컬럼0) → businessNumber 필드
- [ ] 연도(컬럼1) → year 필드
- [ ] 지원사업명(컬럼2) → programName 필드
- [ ] 지원과제명(컬럼3) → projectName 필드
- [ ] 상태(컬럼4) → status 필드
- [ ] 선정금액(컬럼5) → selectedAmount 필드
- [ ] 지원금액(컬럼6) → supportAmount 필드
- [ ] 비고(컬럼7) → notes 필드

### 3. 상태 값 검증
- [ ] 유효한 값: "선정", "완료", "포기", "제외" → 정확히 저장
- [ ] 무효한 값: 기타 문자열 → 기본값 "선정"으로 저장
- [ ] 대소문자 구분: "선정" vs "정선" 등
