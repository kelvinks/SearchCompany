# 자주 묻는 질문 (FAQ)

## Q1. 기업 등록과 지원사업 등록 화면이 다른가요?
**A.** 아니요, 두 화면은 **완전히 동일한 UI 구조**입니다. 탭, 버튼, 프로그레스 바, 미리보기 테이블, 알림 메시지 위치까지 모두 동일합니다. 단지 미리보기 테이블의 **컬럼 구성**과 **데이터 처리 로직**만 다릅니다.

## Q2. 지원사업 등록 시 기업명이 없는데 어떻게 기업을 찾나요?
**A.** 엑셀 파일의 **사업자등록번호** 컬럼을 읽어, 데이터베이스의 `companies` 테이블에서 해당 사업자번호를 가진 기업의 `id`(company_id)를 조회합니다. 찾은 기업명이 미리보기 테이블의 '기업명' 컬럼에 자동 표시됩니다.

## Q3. 매칭되는 기업이 없으면 어떻게 되나요?
**A.** 미리보기 테이블에서 해당 행에 **"(매칭 기업 없음: 사업자번호)"** 형태로 표시되며, 업로드 진행 시 해당 행은 **자동으로 제외**됩니다. 사용자에게는 "X건은 매칭 기업이 없어 등록되지 않습니다"라는 안내가 나옵니다.

## Q4. 기업 등록 시 중복 사업자번호가 있으면 어떻게 처리하나요?
**A.** 업로드 전 미리보기에서 중복 행을 **"중복(업데이트)"** 또는 **"중복(건너뛰기)"** 로 상태를 선택할 수 있습니다. 기본값은 "업데이트"입니다.

## Q5. 필수 필드가 비어있으면 업로드가 안 되나요?
**A.** 네. 필수 필드(기업등록: 기업명/사업자번호, 지원사업: 연도, 지원사업명, 상태)가 비어 있는 행은 미리보기에서 **"필수값 누락"** 상태로 표시되며, 업로드 진행 시 자동 제외됩니다.

## Q6. 엑셀 파일의 컬럼 순서가 바뀌어도 되나요?
**A.** 현재는 **고정 컬럼 순서**를 사용합니다. 템플릿 파일의 컬럼 순서를 변경하지 않고 사용해 주세요.
- 기업 등록: 기업명, 사업자등록번호, 소재지, 주요 사업(지원분야), 주요 제품
- 지원사업: 사업자등록번호, 연도, 지원사업명, 지원과제명, 상태, 선정금액, 지원금액, 비고

## Q7. 업로드 후 결과를 파일로 받을 수 있나요?
**A.** 현재는 화면의 Toast 알림으로만 결과를 보여줍니다. 상세 결과 파일 다운로드 기능은 추후 추가 예정입니다.

## Q8. 한 번에 몇 건까지 업로드 가능한가요?
**A.** 현재 템플릿 기준으로 **최대 1,000건**까지 권장합니다. 그 이상인 경우 여러 파일로 나누어 업로드해 주세요.

## Q9. 실수로 잘못된 데이터를 올렸는데 취소할 수 있나요?
**A.** 업로드 확정("업로드 진행" 버튼 클릭) 전에는 언제든 "다시 선택" 버튼으로 파일을 교체하거나 모달을 닫아 취소할 수 있습니다. 확정 후에는 데이터베이스에 반영되므로 관리자 문의가 필요합니다.

## Q10. 지원사업 등록 시 상태값을 잘못 입력했는데 어떻게 되나요?
**A.** "선정", "완료", "포기", "제외" 4개 값 중 하나가 아니면 기본값 **"선정"**으로 자동 저장됩니다. 미리보기에서 상태값을 확인하고 수정 후 업로드하세요.
